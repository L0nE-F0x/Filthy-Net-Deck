How it works

  1. Daily meta bundle A JSON feed ranks eight decks per format for Bo1 and Bo3 (64 shells per mode).
  2. Scryfall card data Card images and identity resolve from Scryfall’s public API.
  3. Tournament signals Paper RCQs via Melee.gg, metagame % via MTGGoldfish, plus Wizards/Arena notes — always attributed. Desktop app never scrapes; only the daily pipeline does.
